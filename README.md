# betapawa SolarGuard — Landing Page

Solar finance platform landing page for betapawa SolarGuard.

## Stack
- Pure HTML/CSS/JS — zero dependencies, zero build step
- Fonts loaded from Google Fonts CDN
- Auto-deploys to Vercel on every push to `main`

## Deployment
This repo is connected to Vercel. Every push to `main` triggers an automatic deployment.

**Live URL:** https://landing-page-solarguard.vercel.app

## Local Preview
Just open `index.html` in any browser — no server needed.

## Structure
```
index.html      # Full landing page (single file)
vercel.json     # Vercel deployment config
```

## Sections
1. Hero — dual CTA (Finance My Solar / Join as Installer)
2. Customer section — loan calculator + 4-step journey
3. Installer section — 3-track framework + earnings card + disbursement flow
4. How It Works — tabbed (Customer / Installer journey)
5. Testimonials
6. Why SolarGuard
7. Lender partners
8. Dual CTA footer
